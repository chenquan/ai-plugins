## 🎯 核心功能

### 1. 智能审查策略

- **快速审查** (5-10分钟): 日常代码更改
- **深度审查** (30-60分钟): 重要功能和架构
- **培训审查** (60-90分钟): 团队提升和教学
- **健康检查** (全面评估): 项目质量评估

### 2. 专家建议系统

- **即时修复** (0-1周): 具体问题的解决方案
- **预防措施** (1-4周): 工具和流程改进
- **能力提升** (1-3月): 培训计划和知识传递
- **战略规划** (3-12月): 长期改进路线图

### 3. 深度分析能力

- 根本原因分析
- 问题模式识别
- 架构洞察
- 趋势分析

### 4. 进度跟踪

- 问题跟踪系统
- 指标度量
- 定期回顾机制

## 🎓 适用场景

### ✅ 最适合

- 重要功能开发完成时
- 架构级变更后
- 定期质量检查（月度/季度）
- 团队有新人时
- 发现系统性问题时
- 启动质量改进计划时

- 简单的一次性检查（使用 `/cr` 即可）

- 个人学习项目
- 文档修改

## 🔧 配置和定制

### 修改审查规则

编辑 `agents/review-config.json`:

```json
{
  "rules": {
    "code_quality": {
      "max_line_length": 100,
      "max_function_length": 50
    }
  }
}
```

### 添加自定义模板

在 `templates/` 目录添加新的模板文件

### 扩展知识库

编辑 `agents/code-review-knowledge-base.md` 添加新的最佳实践
