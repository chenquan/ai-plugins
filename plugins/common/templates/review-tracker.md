## 问题详情

### CR-001: SQL 注入漏洞

**基本信息**
- **ID**: CR-001
- **标题**: SQL 注入漏洞
- **严重性**: 🔴 P0
- **类别**: 安全性
- **文件**: database.go
- **行号**: 45

**问题描述**
直接拼接用户输入到 SQL 查询，存在 SQL 注入风险

**当前代码**
```go
query := "SELECT * FROM users WHERE id = " + userID
rows, err := db.Query(query)
```

**修复建议**
```go
query := "SELECT * FROM users WHERE id = ?"
rows, err := db.Query(query, userID)
if err != nil {
    return fmt.Errorf("query user: %w", err)
}
defer rows.Close()
```

**需要添加的测试**
```go
func TestQueryUser_SQLInjection(t *testing.T) {
    tests := []struct {
        name    string
        userID  string
        wantErr bool
    }{
        {"normal", "123", false},
        {"injection", "1' OR '1'='1", true},
    }
    for _, tt := range tests {
        t.Run(tt.name, func(t *testing.T) {
            _, err := db.QueryUser(tt.userID)
            if (err != nil) != tt.wantErr {
                t.Errorf("QueryUser() error = %v, wantErr %v", err, tt.wantErr)
            }
        })
    }
}
```
