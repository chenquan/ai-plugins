# 示例 Skill

这是一个示例 Skill，展示了如何创建可复用的专业知识包。

## Skill 特点

- **渐进式加载**：只在需要时加载，不影响主上下文
- **专业知识**：包含特定领域的最佳实践
- **可执行脚本**：可以包含自动化脚本

## 使用场景

当遇到以下情况时，此 Skill 会自动加载：
1. 用户询问相关领域的问题
2. 需要执行特定的专业任务
3. 涉及到最佳实践的建议

## 知识内容

### 核心概念

这里定义核心概念和术语。

### 最佳实践

1. **实践一**：详细说明
2. **实践二**：详细说明
3. **实践三**：详细说明

### 常见问题

**问题一**：如何使用？
- 解决方案...

**问题二**：遇到错误怎么办？
- 排查步骤...

## 创建新的 Skill

1. 在 `skills/` 目录下创建新文件夹
2. 在文件夹中创建 `SKILL.md` 文件
3. 添加 frontmatter（description 等）
4. 编写专业知识和最佳实践
